Documentation for TFC Tweaker:

TFCTweaker add a Crafttweaker Bracket for TFC Metals

Brackets Properties:

Property Type Description
name String The Metal name(e.g copper)
tier int The Metal tier
hasTool boolean true if the metal have tools
hasArmor boolean true if the metal have armor
color int Color of the metal
metlTemp float Melting Temperature of the metal
heat float The Metal Heat
